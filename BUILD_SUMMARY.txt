CRITICAL NOTES

1. NO animation-fill-mode: both on hero content
   - All content is VISIBLE by default
   - Scroll reveal hides then shows on intersection

2. IntersectionObserver scroll reveal implementation
   - Elements visible even if JavaScript fails
   - 15% threshold for smooth reveal

3. RTL Support for Arabic
   - Automatic dir="rtl" on html element
   - All flex layouts work bidirectionally

4. Relative paths only (no absolute paths)
   - All links use relative references (./page.html, #section)

5. LocalStorage data structure
   - static_user: Current logged-in user
